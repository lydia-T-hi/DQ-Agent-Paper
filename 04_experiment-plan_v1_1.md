## 2. 데이터셋 명세

### 2.1 S1 — 국내 도메인 합성 고객 데이터 (주실험)

- 규모: 1,000행 × 12열 (파일럿 스키마 확장)
- 스키마: 고객ID, 이름(한글), 나이, 성별, 이메일, 전화번호, 우편번호, 가입일,
  최근구매일, 구매액(KRW), 등급, 해지여부
- 생성: Faker(ko_KR) + 규칙 기반 교차필드 제약(가입일 ≤ 최근구매일, 등급-구매액 정합)
- 오염 통제 근거: 값 자체가 생성 시점에 신규 → LLM 사전학습 포함 불가능
- 한국어 필드 포함 → 영문 편향(ToV) 완화 + KCI 독자 적합성

### 2.2 S2 — 2025년 이후 원천 기반 정형 셋 (주실험)

- 후보: 2025~2026 공개된 공공데이터포털/통계청 원천 1종을 스키마만 차용,
  값은 재생성(합성) — 실험 착수 전 최종 선정, 선정 기준: 수치+범주 혼합, 10열 이상
- 역할: S1과 도메인이 다른 두 번째 사례 → "합성 셋 1개" 비판 완화

### 2.3 E2 — Hospital, Flights (Raha 표준 벤치마크)

- ground truth 라벨 제공분 사용, 오염 가능성 명시 단서 부착 ("F1 상한 추정치")

## 3. 오류 주입 프로토콜 (`tools/inject_errors.py`)

### 3.1 오류 유형 5종 — R1~R8 규칙과 대응

| 유형 | 조작 | 대응 규칙 | 예 |
|---|---|---|---|
| T1 결측 | 값 → null/""/"N/A"/"-" (센티널 4종 균등) | R1, R2 | 이메일 → "" |
| T2 형식 | 패턴 파괴 | R3, R4 | 전화번호 하이픈 제거, 날짜 포맷 교란 |
| T3 범위 | 극단값·불가능값 치환 | R5, R6 | 나이 → 999 / -1, 구매액 → 10^12 |
| T4 중복 | 행 복제(키 변형 포함) | R7 | 고객ID 접미사 변형 복제 |
| T5 교차필드 | 제약 위반 | R8 | 가입일 > 최근구매일 |

### 3.2 주입 규칙

- 오류율 ρ ∈ {10%, 20%} — 셀 기준, 유형별 균등 배분(각 ρ/5)
- 시드 s ∈ {1..5} 고정 → 반복 5회 = 서로 다른 오류 배치 5개 (동일 배치 재실행이 아님
  — LLM 비결정성과 오류 배치 변동을 함께 포괄, Consistency는 E1과 별도로 s=1 배치
  고정 5회 재실행으로 측정)
- 주입 로그 = ground truth: `{row, col, type, original, corrupted}` JSON 저장

## 4. 구성(A1~A5) 및 공정성 프로토콜

- A1: Profiler+2A / A2: Profiler+2B(전 레코드) / A3: +선택 호출 / A4: +Validator / A5: +HC
- A2 공정성: 2B와 동일 few-shot k=3, 배치 프롬프팅(20행/호출), 동일 출력 스키마 —
  프롬프트 전문 부록 A 공개
- 모델 고정: Detector-LLM = claude-sonnet-4-6, Validator = gpt-4o (버전 문자열 로그에 기록),
  temperature=0, max_tokens 통일

## 5. 지표·검정 (사전 고정)

- **F1**: 셀 단위 오류 탐지 (TP = 주입 셀을 플래그). 유형별 F1 분해 병기
- **Consistency**: s=1 배치 고정 × 5회 재실행, 셀 판정 일치율(Fleiss' κ 보조)
- **Cost**: 토큰(주) — 스테이지별 입출력 합산 / USD(보조) — 실행일 공시 단가
- **CEI**: (F1_k − F1_{k−1}) / (Cost_k − Cost_{k−1}), 분모 토큰·USD 이중 산출
  - 경로 주: A1→A3→A4→A5 / 보조: A1→A3→A3+HC→A5 / ΔCost<ε 규칙 적용
- **검정**: 구성쌍 F1 차이 — Wilcoxon signed-rank (시드 5개 쌍대), α=0.05,
  Cliff's delta 병기. 다중비교는 Holm 보정
- **가설 사전 등록**: H1: F1(A3)>F1(A1) / H2: F1(A5)≥F1(A3) / H3: CEI(2B증분)>CEI(3A증분)
  — 기각되어도 그대로 보고 (v4 금지 표현 준수)
