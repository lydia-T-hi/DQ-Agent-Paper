## 3. 중요한 계측 발견 — CLI 출력 토큰에는 thinking이 포함됨

CLI 감소 폭(118K→80K, 32%)이 예상(~90%)보다 작은 원인: **CLI 경유 시 모델의
thinking 토큰이 output_tokens에 합산**됨 (호출당 ~12-14K). 에코 제거분은 사라졌고
잔여는 대부분 thinking. 함의:

1. **API 백엔드는 thinking 미사용** → 호출당 출력 ~2-4K 전망, A3 회당 **~$0.3-0.5**
   (실측은 크레딧 충전 후)
2. **논문의 비용 지표는 API 백엔드 실측으로 산출해야 함** — CLI 실측 토큰은
   thinking이 포함되어 비용을 과대 계상함. 오늘까지의 USD 수치는 스모크 참고용
3. E1 예산 재추정: 정가 ~$50-90 (기존 $550-650 → 스키마 v3 + thinking 미포함 효과)

## 4. 동결 상태
